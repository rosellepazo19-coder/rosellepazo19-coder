## ğŸ¥¤ Container Beverages Classification App

The **Container Beverages Classification App** is a Flutter-based mobile application designed to identify various beverage containers using **image classification technology**.  
This project was developed as part of the **IT108 Final Project**, integrating **Machine Learning** through **Teachable Machine** and **TensorFlow Lite**.

The application allows users to **scan containers using the device camera** and automatically classify them based on trained container categories.
